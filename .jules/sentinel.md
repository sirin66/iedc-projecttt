## 2026-07-02 - Hardcoded Secrets in Frontend Code
**Vulnerability:** Found multiple hardcoded `atob()` encoded secrets directly inside `app.js` and `admin.html` including Admin passwords, gate passwords, Firebase config (API key, project id, etc.), Supabase credentials and EmailJS keys.
**Learning:** Hardcoded encoded secrets `atob()` can be trivially decoded and expose sensitive configuration endpoints and administrative passwords to any end-user inspecting the frontend codebase. In an application without a module bundler or separate backend, secrets were hardcoded directly in client-side code instead of loading them at runtime or keeping them in a backend.
**Prevention:** Avoid putting admin passwords, keys and secrets in the frontend bundle. Utilize externalized configurations such as an environment specific configuration file that is loaded at runtime (e.g. `config.local.js`) and excluded from source control for environments, or handle authentication and business logic completely server-side.
